爱个购商户使用协议
    本《爱个购商户使用协议》由公司定期更新，更新后的《爱个购商户使用协议》条款一旦公布即代替原来的 《爱个购商户使用协议》条款，恕不再另行通知，甲方有义务不时关注并阅读最新版的《爱个购商户使用协议》条款及“爱个购”官网、微信公众号、“爱个购”客 户端的公告，在公司修改《爱个购商户使用协议》条款后，如果用户不接受修改后的条款，应立即停止接受本公司提供的服务，如用户继续使用公司提供的服务将被 视为已接受了修改后的《爱个购商户使用协议》。

    甲、乙双方均是按照中华人民共和国法律成立并存续的公司或个体工商户，如无经营资格或超越其民事权利或 行为能力范围从事交易的，本协议自始无效，一经发现，守约方有权立即终止本协议，并追究违约方一切法律责任，甲方为合作商家，乙方是爱个购指定区域代理， 甲乙双方经友好协商，就甲方通过乙方所代理的爱个购平台进行网络营销推广事宜，在平等自愿的基础上达成如下协议，双方共同遵守：

一、 合作内容

（一）积分返利规则：

  • 1.1 积分返利是指甲方从用户每次消费返利金额中拿出消费金额的18%作为返利支付给受益人，并充值消费的相应金额用于支付受益者积分。受益人包括有五方：爱个 购平台所有人、区域代理、用户、用户的直接推荐人、用户直接推荐人的直接推荐人，返利的支付义务人是甲方，甲方在爱个购手机端商户入口内进行在线充值，可 以获得等值的积分（1积分=1元人民币），爱个购平台通过扣除积分的形式对返利进行等值量化处理，积分返利按照比例分配给受益人.爱个购平台保留对积分返 利在用户消费实际支付金额中所占比例的最终修改权，若有修改，乙方通知甲方进行协商.爱个购通过积分返利，促使更多的用户去分享和消费，商家获得更多的粉 丝用户，可以更有效的做好宣传推广，吸引更多的新客户，维护老客户，提高营业额，增加盈利。
  • 1.2 积分返利规则：
  • 1.2.1 用户每次在爱个购认证商家消费时直接获取商家提供的18%比例积分返利。
  • 1.2.2 用户每次在爱个购认证商家消费时，其直接推荐人可以获取该商家提供的18%比例积分返利。
  • 1.2.3 用户每次在爱个购认证商户消费时，其直接推荐人的直接推荐人可以获取商家提供的18%比例积分返利。
  • 1.2.4 爱个购平台区域代理，在用户至该区域爱个购商家消费时，可获取商家提供的18%比例积分返利。
  • 1.2.5 在用户到爱个购商家消费时，爱个购平台可获取商家提供的10%比例积分返利。
  • 1.2.6 甲方同样可以邀请消费者注册成为爱个购用户，甲方作为推荐者，同样可以获取1.2.2及1.2.3所述的积分返利。
  • 1.2.7 甲方未及时确认用户消费金额，或者积分不足导致不能及时返利，导致受益人投诉时，甲方需支付受益人双倍积分返利。
  • 1.2.8 甲方不提供甲方发布在爱个购平台的优惠活动给用户，造成用户投诉，甲方需支付用户10积分。
  • 1.2.9 甲方同意各受益人之间获取积分返利的比例由爱个购平台所有人决定。
  • 1.3 积分返利规则：
  • 1.3.1 甲方可以随时在爱个购手机端中用积分提取等值的现金，3~5个工作日内到账。
  • 1.3.2 用户可以在爱个购手机端中用积分提取等值的现金，3~5个工作日内到账。

（二）关注用户管理：

  • 2.1 关注用户是指用户在爱个购微信公众号和App中关注甲方即可成为甲方的关注用户。
  • 2.2 关注用户管理的方式：通过后台中“关注用户管理”项目进行了解用户信息和进行管理。

（三）消息推送：

  • 3.1 免费推送消息：每天最多能向关注用户推送一次App推送消息，推送消息不得有反动、暴力、黄色、虚假、欺骗等不合法内容及与甲方经营范围无关的内容，否则所引起的一切版权或法律纠纷由甲方自行承担。
  • 3.2 精准推送消息：指甲方利用爱个购后台管理系统提供的大数据服务给特定消费者进行推送App消息.精准推送信息需甲方支付佣金给爱个购平台，甲方每发送一条 精准推送信息需支付壹元人民币/人/条的佣金，受益人是爱个购平台所有人和乙方。 甲方同意各受益人之间获取精准推送佣金的比例由爱个购平台所有人决定。

（四）订单确认协议：

  • 商家使用商户APP确认订单时，请严格遵守使用规则，并保护好自己的下属员工的手机账号，如下属员工离职，及时删除相关信息，使其手机系统无确认订单权利，如因商户保护账号不周，本公司对其相关损失概不负责。

二、 双方的权利与义务

  • 2.1 甲方的权利与义务
  • 2.1.1 甲方承诺其是本协议项下店铺的真实经营管理者，其在爱个购官网和爱个购后台管理系统注册时提交的身份证、营业执照等所有证明材料合法、真实、准确详尽，否则甲方应承诺独立承担因此导致的相应责任及后果，使乙方和爱个购平台所有人免责。
  • 2.1.2 甲方承诺其严格遵守在爱个购官网注册时，阅读并同意的商户注册协议。
  • 2.1.3 甲方承诺不与乙方代理区域以外的爱个购代理商签约。
  • 2.1.4 甲方承诺遵守《网络交易管理办法》、《消费者权益保护法》、《反不正当竞争法》等法律法规，诚信经营，合法经营，依法纳税，不欺诈消费者，不售卖假冒伪劣产品。因甲方服务质量等原因引起的消费纠纷，包括赔偿、法律诉讼等，甲方独立承担所有责任。
  • 2.1.5 甲方有义务在客户消费产生后给客户提供发票、小票等消费凭据。
  • 2.1.6 甲方承诺保证账户积分充足，及时确认客户提交的消费信息，按时足额支付返利和精准推送佣金。因积分返利和精准推送返佣支付不及时不到位对受益人造成的违约责任和赔偿责任，由甲方独立承担。
  • 2.1.7 甲方承诺其通过爱个购后台管理系统发布、编辑、更新的商户和商品信息真实准确，并且不侵犯任何人的合法权益。
  • 2.1.8 在本协议有效期内及本协议终止或者解除后，甲方承诺严格保守乙方的商业机密，若甲方泄露相关商业机密的，乙方有权追究相关的法律责任。
  • 2.2 乙方的权利与义务
  • 2.2.1 乙方有责任对甲方进行爱个购后台管理系统的使用培训，解答甲方提出的问题，处理甲方提出的建议，协助甲方策划营销方案。
  • 2.2.2 爱个购平台所有人对积分返利比例、精准推送佣金等涉及甲方的相关佣金比例做调整时，乙方有责任在接到调整一周内，通知甲方，进行沟通协商。
  • 2.2.3 乙方有权对甲方在爱个购官网和爱个购后台管理系统注册时提交的证明材料进行审核。
  • 2.2.4 乙方协助爱个购平台给甲方提供认证服务，也有权在甲方出现违规行为、违约行为、被客户投诉以及其他不良情形下，降低甲方认证等级或者取消对甲方的认证。
  • 2.2.5 乙方有权针对用户的投诉向甲方了解情况，做出投诉处理结果，用户持有的消费凭据或者用户在消费现场拍摄的照片或摄像将作为处理投诉的依据。
  • 2.2.6 乙方不得未经爱个购平台同意以任何名义和理由向甲方直接收取费用。
  • 2.2.7 在本协议有效期内及本协议终止或者解除后，乙方承诺严格保守甲方的商业机密，若乙方泄露相关商业机密的，甲方有权追究相关的法律责任。

三、 协议的终止

  • 3.1 本协议一方有权在提前一周通知对方的情况下单方面解除本协议，终止合作。
  • 3.2 任何一方违反或者不履行本协议中己方项下的义务，对方有权立即解除本协议，终止合作。
  • 3.3 双方终止合作后，应对合作期内甲方应付返利和精准推送返佣、甲方账户积分余额等事宜进行结算。3.2项终止合作的，违约方还应承担违约责任和赔偿责任。
  • 3.4 双方一致同意，如甲方违反本协议第二条的甲方义务，爱个购平台所有人有权采取一切必要的措施，包括但不限于：删除甲方发布的内容、取消甲方获得的认证、扣罚甲方账户积分，暂停或查封甲方帐号，取消甲方因违规所获利益，终止本协议等。
  • 3.5 自本协议终止之日起，甲方的认证商户身份即行终止，甲方在爱个购平台的账户、爱个购后台使用权限和爱个购商户版App使用权限将予以关闭，甲方发布的电子优惠券以及甲方在爱个购平台发布的其他信息将予以清除。

四、 特别约定

  • 4.1 双方确认：爱个购平台所有人有权随时调整爱个购官网和爱个购后台管理系统商户注册协议中的内容，双方遵守调整后的商户注册协议。
  • 4.2 因履行本协议发生的纠纷，由甲乙双方处理，并由本协议签订地——爱个购平台所有人所在地人民法院管辖，诉讼不得牵涉爱个购平台所有人，由甲乙双方各自独立承担责任。
  • 4.3 任何一方对本协议的内容和对方当事人的商业机密均负有保密的义务。
  • 4.4 本协议未尽事宜，由双方协商一致，签署补充协议，与本协议具有同等法律效力。
  • 4.5 本协议中所出现的各方地址、电话、电子邮箱、QQ、微信号为双方发出通知和接受通知的有效联系方式和通讯地址。一方如变更通讯地址或联系方式，应自变更之日起二日内，将变更后的地址、联系方式通知另一方，否则变更方应对此造成的一切后果承担责任。
  • 4.6 甲方若是连锁经营，甲乙双方需协助分支机构与其所在区域对应的爱个购区域代理合作事宜。
  • 4.7 本协议自双方在本协议签字盖章之日起生效。本协议一式三份，甲方乙方各执一份，另交一份给爱个购平台所有人备案。
